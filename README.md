# CWG Dashboard - NLCB Charts & Analysis

## 📦 Installation

### Method 1: Scriptable (Recommended)
1. Copy the code from `loader.js`
2. Open Scriptable on your device
3. Create a new script called "CWG Loader"
4. Paste the code and save
5. Run the script - it will automatically download the dashboard

### Method 2: Direct Download
1. Download `CWG_Dashboard.js`
2. Save to Scriptable's folder
3. Run directly

## 🔄 Auto Updates
The loader checks for updates every time you run it.
You'll see a notification when new features are available.

## 📊 Features
- Hot & Overdue Numbers
- Shelf Marks Analysis  
- Day-to-Day Charts
- Leaving/Meeting Logic
- Pick 2 & Pick 4 Analysis
